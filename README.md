# Сайт школи

Офіційний сайт школи, створений на **Vue 3** та **Vite**.
Проєкт містить головну сторінку, інформаційні розділи, сторінку контактів, блоки з новинами/фото та адаптивну верстку для комп’ютерів і мобільних пристроїв.

## Опис проєкту

Цей сайт призначений для зручного представлення інформації про навчальний заклад:

* головна сторінка з вітальним блоком;
* інформація про школу;
* сторінка контактів;
* телефони та електронна пошта;
* фотоматеріали;
* адаптивний дизайн;
* підготовка до розміщення на GitHub Pages.

## Технології

У проєкті використовуються:

* **Vue 3**
* **Vite**
* **JavaScript**
* **HTML**
* **CSS**
* **GitHub Pages**

## Встановлення

Склонуйте репозиторій:

```bash
git clone https://github.com/InterGamesInt/School60Site
```

Перейдіть у папку проєкту:

```bash
cd School60Site
```

Встановіть залежності:

```bash
npm install
```

## Запуск у режимі розробки

```bash
npm run dev
```

Після запуску сайт буде доступний у браузері за локальною адресою, яку покаже Vite, наприклад:

```text
http://localhost:5173/
```

## Збірка проєкту

Для створення production-версії сайту виконайте:

```bash
npm run build
```

Готові файли будуть створені у папці:

```text
dist
```

## Попередній перегляд production-збірки

```bash
npm run preview
```

## Публікація на GitHub Pages

Якщо в проєкті налаштований деплой через `gh-pages`, сайт можна опублікувати командою:

```bash
npm run deploy
```

Або одразу зібрати та опублікувати:

```bash
npm run build:deploy
```

## Структура проєкту

Приблизна структура:

```text
src/
├── assets/          # Зображення та інші ресурси
├── components/      # Компоненти сайту
├── pages/           # Сторінки сайту
├── router/          # Налаштування маршрутів
├── App.vue          # Головний компонент
└── main.js          # Точка входу
```

На сторінці контактів вказані телефони та електронна пошта навчального закладу.

Приклад телефонів:

```text
+38 (044) 275-51-91
+38 (044) 275-74-54
```

## Примітки щодо роботи посилань

Посилання типу `mailto:` відкривають поштову програму або веб-пошту користувача.

```html
<a href="mailto:example@example.com">example@example.com</a>
```

Посилання типу `tel:` відкривають набір номера на телефоні або програму для дзвінків.

```html
<a href="tel:+380442755191">2755191</a>
```

На комп’ютері такі посилання можуть не працювати, якщо не налаштована програма для пошти або дзвінків за замовчуванням.
